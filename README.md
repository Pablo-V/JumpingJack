# Jumping Jack
#### Controles:
- A: Izquierda
- D: Derecha
- Barra espaciadora: Salto
#### Cómo iniciar:
Descargar el archivo **desktop.jar** situado en el directorio raíz de este repositorio y ejecutar. Se necesita Java 8 y versiones posteriores para poder jugar.
#### Código base:
El código base del juego está localizado en la carpeta **/core/src/com/pablov/jumpingjack**, disponible para quien quiera utilizarlo.
#### Cómo jugar:
El objetivo del juego consiste en alcanzar el final del nivel sin ser derrotado por obstáculos o enemigos, y continuar avanzando sin perder vidas para no perder progreso.
Controlas a Jack, un alienígena perdido en el planeta Tierra, el cual intenta buscar su nave para volver a casa. Para ello, deberá hacerse paso a través de zonas llenas de enemigos y obstáculos. Jack tiene unas capacidades de salto extraordinarias, pudiendo saltar tres metros con un solo salto y también puede saltar encima de muelles para tener un gran impulso hacia arriba. Además, sus pies y cabeza son muy resistentes. Su cabeza es capaz de destruir bloques, y sus pies pueden derrotar a cualquier enemigo al caer encima, como los ratones. Los bloques simples otorgarán una cantidad pequeña de puntos, los bloques con monedas darán una cantidad intermedia de puntos, y los bloques sorpresa "!" le darán una vida extra a Jack. Además, derrotar enemigos proporcionará una gran cantidad de puntos.
#### Dónde está mi partida guardada?
La partida guardada se puede encontrar en C:\Usuarios\**nombre_de_usuario*\.prefs, el archivo se llama "datos_jumpingjack". Este archivo puede utilizarse en ordenadores diferentes siempre que se coloque en el mismo directorio.
