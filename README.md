# Jumping Jack
#### Controles:
- A: Izquierda
- D: Derecha
- Barra espaciadora: Salto
#### Cómo iniciar:
Descargar el archivo **desktop.jar** situado en el directorio raíz de este repositorio y ejecutar. Se necesita Java 8 y versiones posteriores para poder jugar.
#### Código base:
El código base del juego está localizado en la carpeta **/core/src/com/pablov/jumpingjack**, disponible para quien quiera utilizarlo.
